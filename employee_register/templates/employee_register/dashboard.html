ABOUT THE SYSTEM

Add Employee

This operation provides a form to input the details of an employee and stores the entered information to the database (DB.sqlite) for that employee.

View Employee List

This operation presents a list of all the employees in the database. The list is presented as a row with each row entitled to a unique person and at the end of row of first table, there is a update and delete option.

Search

This operation presents a list of all employees just like the View Employee List but helps admin search employees or even a particular employee(if the employee ID is known)
